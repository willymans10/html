# HTML Style and Colors

Materi kali ini akan membahas tentang cara mempercantik laman website kamu. Kamu akan mempelajari HTML Style Attribute,Background Color, Text Color, Text Size, Fonts dan Border Color.

Setelah mempelajari materi ini, kamu akan dapat membuat website kamu lebih berwarna mencerminkan pesan yang ingin kamu bawakan.

## Atribut Gaya HTML (HTML Style Attribute)

**HTML Style Attribute** digunakan untuk menambahkan sebuah style pada sebuah elemen. Contohnya color, font, size, dan yang lainnya

### Struktur Dasar Atribut Gaya HTML (Basic HTML Style Attribute Structure)

```
<tag_name style="property_name: value;">Content</tag_name>

<nama_tag style="nama_property: nilai;">Isi konten</nama_tag>

<p style="color: #fff;">Hello World</p>
```

### Warna Latar (Background Color)

**Background Color** adalah style yang digunakan untuk mengubah warna background setiap elemen HTML.

Property yang digunakan adalah: **background-color**

- Contoh penggunaan background color

```
<tag_name style="property_name: value;">Content</tag_name>

<nama_tag style="nama_property: nilai;">Isi konten</nama_tag>

<div style="background-color: green;"></div>
```

### Warna Tulisan (Text Color)

**Text Color** adalah style yang digunakan untuk memberikan warna pada text atau tulisan.

Property yang digunakan adalah: **color**

- Contoh penggunaan text color

```
<tag_name style="property_name: value;">Content</tag_name>

<nama_tag style="nama_property: nilai;">Isi konten</nama_tag>

<p style="color: red;">Hello World</p>
```

### Ukuran Text atau Huruf (Text Size/Font Size)

**Text Size** adalah style yang digunakan untuk mengatur ukuran dari sebuah text atau tulisan

Property yang digunakan adalah: **font-size**

- Contoh penggunaan font size

```
<tag_name style="property_name: value;">Content</tag_name>

<nama_tag style="nama_property: nilai;">Isi konten</nama_tag>

<p style="font-size: 20px;">Hello World</p>
```

### Gaya Huruf (Font Style)

**Font Style** adalah style yang digunakan untuk mengatur tampilan atau gaya dari huruf

Property yang digunakan adalah: **font-family**

- Contoh penggunaan font style

```
<tag_name style="property_name: value;">Content</tag_name>

<nama_tag style="nama_property: nilai;">Isi konten</nama_tag>

<p style="font-style: arial;">Hello World</p>
```

### Border

**Border** adalah style yang digunakan untuk memberikan garis batas pada elemen HTML

Property yang digunakan adalah: **border**

- Contoh penggunaan border 1 kondisi

```
<tag_name style="property_name: value;">Content</tag_name>

<nama_tag style="nama_property: nilai;">Isi konten</nama_tag>

<p style="border: solid;">Hello World</p>
```

- Contoh penggunaan border 2 kondisi

```
<tag_name style="property_name: value;">Content</tag_name>

<nama_tag style="nama_property: nilai;">Isi konten</nama_tag>

<p style="border: 2px solid;">Hello World</p>

<p style="border: solid red;">Hello World</p>
```

- Contoh penggunaan border 3 kondisi

```
<tag_name style="property_name: value;">Content</tag_name>

<nama_tag style="nama_property: nilai;">Isi konten</nama_tag>

<p style="border: 2px solid red;">Hello World</p>
```
